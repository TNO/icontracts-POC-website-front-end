# Hosting

## Locally

```Use node 16```

command:
```nvm node 16```

```npm run serve ```

or

```Npm run build```

## using Docker

Use 
```docker-compose up --build```
and other docker commands together with https://github.com/onassisontology/icontracts_back_end/blob/main/docker-compose.yml
